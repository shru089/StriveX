## 🎨 Customization

Want to change colors? Edit `frontend/css/style.css`:

```css
:root {
    --primary-gradient: linear-gradient(135deg, #YOUR_COLOR_1, #YOUR_COLOR_2);
    --secondary-gradient: linear-gradient(135deg, #YOUR_COLOR_3, #YOUR_COLOR_4);
}
```
